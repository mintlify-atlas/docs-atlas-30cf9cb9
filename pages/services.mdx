The services page is registered at `/servicios` and rendered by the `servicios_page` template. It is accessible from the main navigation menu (sequence 10).

| Field | Value |
|---|---|

| Heading | "Soluciones tecnológicas a la medida de tu empresa" |
| Subheading | Overview of the four service areas |

## Service rows

Each service is presented as a full-width alternating row (`zw-service-row`). Even-numbered rows carry the `zw-service-row--reverse` modifier, placing the image on the right.

## Route details

```python
@http.route('/servicios', type='http', auth='public', website=True)
def servicios(self, **kwargs):
    return request.render('zonaweb_website.servicios_page')
```

The route is public and website-aware. No authentication is required.
