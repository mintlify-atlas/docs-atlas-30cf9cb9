## CSRF protection

The form includes a hidden CSRF token rendered server-side by Odoo's QWeb engine:

```xml
<input type="hidden" name="csrf_token" t-att-value="request.csrf_token()"/>
```

The `/submit-bidding` route is declared with `csrf=True`, so Odoo validates the token on every `POST` request. Requests without a valid token are rejected before the controller logic runs.

## Form submission flow

<Steps>
  <Step title="User submits the form">
    The browser `POST`s all field values plus the CSRF token to `/submit-bidding`.
  </Step>
  <Step title="Record creation">
    The `submit_bidding` controller creates a `zonaweb.bidding.request` record via `sudo()`, assigning an auto-generated sequence reference (e.g., `BID/0001`).
  </Step>
  <Step title="Email confirmation">
    The controller attempts to send the `email_template_bidding_confirmation` mail template to the submitter. If sending fails, the error is logged as a warning and the flow continues.
  </Step>
  <Step title="Thank-you page">
    On success, the visitor is redirected to the `bidding_thankyou` template, which displays the reference number (`bidding_id`) returned from the record.
  </Step>
  <Step title="Error handling">
    If record creation itself fails, the controller renders `website.http_error` with a user-facing message.
  </Step>
</Steps>

## Thank-you page

After a successful submission, the `bidding_thankyou` template is rendered with:

| Element | Value |
|---|---|
| Icon | `fa-check` (green) |
| Heading | "¡Solicitud Recibida!" |
| Body | Team will review and contact within 24 hours |
| Reference | The auto-generated `name` field value from `BiddingRequest` |
| Return link | "Volver al Inicio" → `/` |

```python
    return request.render('zonaweb_website.bidding_page')

# Form submission
@http.route('/submit-bidding', type='http', auth='public',
            methods=['POST'], website=True, csrf=True)
def submit_bidding(self, **kwargs):
    ...
```

Both routes are public and website-aware. No authentication is required to view or submit the form.
