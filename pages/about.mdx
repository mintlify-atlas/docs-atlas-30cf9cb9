The about page is registered at `/nosotros` and rendered by the `nosotros_page` template. It is accessible from the main navigation menu (sequence 30).

## Page hero

| Field | Value |
|---|---|
| Label | "Sobre Nosotros" |
| Heading | "Más de 10 años impulsando la transformación digital" |
| Subheading | Brief description of the team's passion for technology |

## Company intro

The intro section uses a two-column grid (`zw-about-intro__grid`): an image (`about-image.jpg`) on the left and text content on the right.

A floating stat overlay on the image displays:

| Field | Value |
|---|---|
| Number | `2013` |
| Label | Año de fundación |

**Narrative text:**
- First paragraph: Zonaweb's mission to democratize high-quality technology solutions, combining technical expertise with business understanding
- Second paragraph: Multidisciplinary team working with the latest technologies to develop scalable, sustainable solutions

## Stats band

A dark-background band (`zw-stats-band`) displays four animated counters. Each `[data-count]` element animates from 0 to its target when scrolled into view (see the counter behavior in `zonaweb_script.js`).

| `data-count` | Suffix | Label |
|---|---|---|
| `10` | `+` | Años de experiencia |
| `120` | `+` | Proyectos completados |
| `85` | `+` | Clientes activos |
| `98` | `%` | Satisfacción del cliente |

<Note>
The homepage hero shows three counters (10+, 120+, 98%). The about page stats band adds a fourth: **85+ clientes activos**.
</Note>

## Bottom CTA

- **Heading:** "¿Quieres unirte a nuestro equipo?"
- **Body:** Open invitation for technology talent
- **Button:** "Contáctanos" → `/contactus`

<Tip>
The about page CTA links to `/contactus` (Odoo's built-in contact form) rather than `/licencias`. This is intentional — the target audience for recruitment inquiries is different from the bidding audience.
</Tip>

## Route details

```python
@http.route('/nosotros', type='http', auth='public', website=True)
def nosotros(self, **kwargs):
    return request.render('zonaweb_website.nosotros_page')
```
