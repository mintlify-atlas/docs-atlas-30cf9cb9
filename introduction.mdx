# Tema Zonaweb

Tema Zonaweb (`zonaweb_website`) is a complete Odoo 18 module that transforms your Odoo instance into a professional corporate website for technology service companies. It provides a fully styled, responsive website with dedicated pages for services, portfolio, team, and a built-in bidding/quote request system.

## Key features

- **Brand colors** — Custom CSS variables (`--primary: #0057FF`, `--secondary: #0B1F3A`, `--accent: #00C2FF`) applied globally across all templates
- **Animated UI** — Hero carousel with auto-play, scroll-reveal animations on cards, and animated counters
- **Bidding workflow** — Four-state lifecycle (Draft → Confirmed → Approved/Rejected) with email notifications and chatter logging
- **JSON API** — A `/api/bidding-status` endpoint for programmatic status lookups
- **Odoo integration** — Inherits `website`, `website_sale`, `mail`, and `sale_management` to slot seamlessly into any Odoo 18 installation
