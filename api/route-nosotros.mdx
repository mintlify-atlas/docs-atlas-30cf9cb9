Returns the about us page rendered with the `zonaweb_website.nosotros_page` QWeb template.

## Endpoint

```
GET /nosotros
```

| Property | Value |
|----------|-------|
| Type | `http` |
| Auth | `public` |
| Website context | yes (`website=True`) |
| CSRF | not applicable (GET) |

## Request parameters

This route accepts no defined parameters. Any extra query string values are captured by `**kwargs` and silently ignored.

## Response

Returns an HTTP 200 response with the rendered HTML page.

| Condition | Response |
|-----------|----------|
| Always | HTML — `zonaweb_website.nosotros_page` template |

## Example

```bash cURL
curl -X GET https://your-odoo-instance.com/nosotros
```

<Note>
Because this is a standard website page, browsers receive the full Odoo website layout (header, footer, assets) as part of the rendered template. The response is not JSON.
</Note>
