`zonaweb_style.css` is the single stylesheet that styles every page of the Zonaweb theme. This page catalogs all CSS variables and the key class patterns you will encounter when customizing or extending the theme.

## CSS custom properties

All variables are declared on `:root` and are therefore available globally.

## Section classes

### `zw-section`

Apply to `<section>` elements to get 100 px vertical padding.

```html
<section class="zw-section"> ... </section>
```

### `zw-section--light`

Pair with `zw-section` to apply the `--off-white` background color.

```html
<section class="zw-section zw-section--light"> ... </section>
```

### `zw-section--dark`

Applies `--navy` background and white text. Used for dark sections on white pages.

```html
<section class="zw-section zw-section--dark"> ... </section>
```

### `zw-section-sm`

Reduced padding variant — 60 px instead of 100 px.

## Animation classes

### Entry animations

Apply these classes to elements that should animate when the page loads.

| Class | Keyframe | Duration |
|---|---|---|
| `animate-fade-up` | `fadeUp` — slides up from 30 px below | 0.7 s |
| `animate-fade-in` | `fadeIn` — opacity from 0 to 1 | 0.6 s |
| `animate-slide-right` | `slideRight` — slides in from 30 px left | 0.7 s |
| `animate-scale-in` | `scaleIn` — scales from 0.92 to 1 | 0.5 s |

### Delay utilities

Combine a delay class with any animation class to stagger siblings.

```html
<div class="animate-fade-up">First item — no delay</div>
<div class="animate-fade-up delay-2">Second item — 0.2 s delay</div>
<div class="animate-fade-up delay-4">Third item — 0.4 s delay</div>
```

| Class | `animation-delay` |
|---|---|
| `delay-1` | 0.1 s |
| `delay-2` | 0.2 s |
| `delay-3` | 0.3 s |
| `delay-4` | 0.4 s |
| `delay-5` | 0.5 s |
| `delay-6` | 0.6 s |

## Scroll reveal (JavaScript)

On page load, `zonaweb_script.js` sets up an `IntersectionObserver` that fades cards in as they enter the viewport. The observer targets five card types:

```javascript static/src/js/zonaweb_script.js
const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            entry.target.style.opacity = '1';
            entry.target.style.transform = 'translateY(0)';
            observer.unobserve(entry.target);
        }
    });
}, { threshold: 0.1 });

document.querySelectorAll(
    '.zw-service-card, .zw-portfolio-card, .zw-pf-card, .zw-plan-card, .zw-team-card'
).forEach((el, i) => {
    el.style.opacity = '0';
    el.style.transform = 'translateY(24px)';
    el.style.transition = `opacity 0.5s ease ${i * 0.08}s, transform 0.5s ease ${i * 0.08}s`;
    observer.observe(el);
});
```

Each card is staggered by `index × 0.08 s`, so cards within the same grid appear sequentially rather than simultaneously. The observer fires once per element (`unobserve` is called after the first trigger).

<Note>
If you add new card components and want scroll reveal applied to them, either add their class to the selector string in `zonaweb_script.js`, or replicate the inline style pattern in your own JS file registered under `website.assets_frontend`.
</Note>

## Typography variables

The theme uses two Google Fonts loaded via `@import` at the top of `zonaweb_style.css`:

| Font | Weights | Role |
|---|---|---|
| **Syne** | 400, 600, 700, 800 | Headings (`h1`–`h6`), stat numbers, plan prices |
| **DM Sans** | 300, 400, 500 (+ italic 400) | Body text, buttons, form labels |

The `font-family` is set on `:root` and `body`, so you can override it with a custom font by redeclaring `font-family` in a later `:root` block or directly on `body`.
